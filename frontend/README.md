# Frontend - Ad-Insight-Spec

Vue.js + Pinia ベースのフロントエンドアプリケーション

## 技術スタック

- **Framework**: Vue.js 3
- **State Management**: Pinia
- **Build Tool**: Vite
- **Styling**: Tailwind CSS（予定）

## ディレクトリ構成

- \src/components/\ - 再利用可能な UI コンポーネント（TODO）
- \src/views/\ - ページレベルのコンポーネント（TODO）
- \src/stores/\ - Pinia ストア（TODO）
- \src/router/\ - Vue Router 設定（TODO）
- \src/services/\ - API クライアント（TODO）
- \public/\ - 静的ファイル
